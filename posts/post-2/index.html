How did I made the website

August 27, 2023 2-minute read

1) Install Hugo Link to heading

sudo pacman -S hugo

2) Create two repo named .github.io and blog Link to heading

3) Clone these two repositories, make a README.md in .github.io and make an initial commit and push. Link to heading

4) Create a hugo site in blog repo. Download the theme you want from hugo themes website and move it to themes directory of the hugo site. Link to heading

5) In hugo.toml, change baseURL to ‘https://.guthub.io/’, give appropriate title and add “theme = ‘’. Link to heading

6) In themes/, edit the config.toml and add the example config from the theme creator and edit accordingly. You can add an image, name, whatever you want. Link to heading

7) To preview the website, start a local server from the root of the site directory. The -w watches filesystem for changes and recreate as needed. And -D make drafts visible. Then when you open the link provided, you can see your website and edit accordingly Link to heading

hugo server -w -D

8) To make posts, add layout of the post in archetypes directory as posts.md and make post using this command: Link to heading

hugo new posts/<post_name>.md

In hugo site directory, create a submodule of main branch of .github.io to a new directory called public. Link to heading

git submodule add -b main <github_repo_link> public

To build your site Link to heading

hugo -t

Push everything to github from blog github clone Link to heading

git add . && git commit -m ‘commit’ -a && git push

This will make the website active Link to heading
